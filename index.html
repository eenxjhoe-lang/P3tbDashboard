☁️ Sinkronisasi Google Sheets

Hubungkan dashboard ke Google Sheets agar data tersimpan terpusat & bisa diakses semua petugas

Belum Terhubung ke Google Sheets
Masukkan URL Apps Script untuk memulai sinkronisasi
💾 Data Lokal
Pengaturan Koneksi Apps Script
💡 URL harus diakhiri /exec — bukan /dev
🔗 Buka URL di tab baru (cek respons JSON) Harus tampil teks JSON, bukan halaman login
🔧 Panduan Mengatasi Gagal Koneksi
1
Cek URL — Pastikan URL diakhiri /exec. Kalau berakhiran /dev, ganti ke /exec
2
Cek Akses — Di Apps Script → Deploy → Kelola Deployment → Edit → pastikan "Siapa yang punya akses: Semua orang" (bukan "Semua orang yang memiliki akun Google")
3
Deploy Ulang — Setelah ganti kode, wajib deploy ulang dengan Versi Baru. Deploy lama tidak akan update kode
4
Tes Manual — Klik tombol "Buka URL di tab baru" di atas. Harus tampil teks JSON seperti {"status":"ok",...}. Kalau muncul halaman login Google → akses masih terbatas
5
Izin Google — Saat pertama deploy, harus klik "Tinjau izin" → "Izinkan". Kalau dilewati, Apps Script tidak akan jalan
Panduan Cara Menghubungkan (5 Langkah)
1
Buka Google Apps Script
Kunjungi script.google.com → Klik "Proyek Baru"
2
Paste Kode Apps Script
Hapus kode default → Paste isi file DOPK_AppsScript_Code.gs → Ubah email di bagian CONFIG
3
Jalankan fungsi cekSetup
Pilih fungsi cekSetup → Klik Run → Izinkan akses Google → Spreadsheet otomatis dibuat
4
Deploy sebagai Web App
Klik Deploy → Deployment baru → Tipe: Web App → Akses: Semua orang → Deploy → Copy URL
5
Paste URL di atas → Tes Koneksi → Simpan
Setelah tersambung, semua data otomatis tersinkron ke Google Sheets 🎉
